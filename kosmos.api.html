Основная информация

Космос (аббр.: Сценарий, Компилируемый в модуль) — язык программирования, который компилируется в javascript. Программы, которые получаются в последствии, называются модулями. Их можно подключить в свою программу без конвертирования, нужно только указать URL файла

История

Язык программирования впервые был применён 20 февраля 2018 года в операционной системе "Фатон 1" в качестве демонстрации возможностей ОС. Он был основан на языке, который был реализован в книге "Выразительный JavaScript" в главе "Проект: язык программирования". Сценарий вводится в кодовой строке, которая указывает на то, что именно там пишется код. Пример кода будет приведён ниже

Основы создания программ

Специальные формы

  • сделать(код) — выполнить какую либо функцию
  • опознать(имя, значение) — создание переменной
  • изменить(имя, значение) — изменить значение переменной на другое
  • функция(аргБескончн, действияБесконечн) — функции с неограниченными аргументами и вызовами других функций
  • если(условие) код — проверка условий
  • цикл(аргументы) — цикличное выполнение

    • Окружение

  • да — верное значение (true)
  • нет — неверное значение (false)
  • массив — массив, хранящий в себе данные
  • длина — длина массива
  • элемент — поиск элемента, который хранится в массиве

    • Встроенные функции

  • вывод(значение) — вывод значения в консоль

    • Создание программ

    * Примечание: В кодовой строке пишите без комментариев (#) и переносов строк. Желательно сплошным текстом, например так: сделать(вывод("А"),вывод(+(1, 2)), опознать(а, * (3, 8)), вывод(а)) Это избавит вас от ошибок ненайдных переменных и обрывов строк. Клавишу Enter нажимайте, когда код будет полностью готов

    Привет, Мир!

    В данном примере программа выводит текстовую строку в Терминал: сделать(
        вывод("Привет, Мир!")
    )
     Результат в консоли: Привет, Мир! Теперь перейдём к переменным

    Переменные

    Создадим программу и добавим переменную А: сделать(
        опознать(А, "Привет, Мир! Ты - переменная"),
        вывод(А)
    )     Добавим любой текст или число к этой переменной: сделать
    (
        опознать(А, "Привет, Мир!"),
        вывод(+(А, " Ты - переменная."),
        вывод(А),
    )     Применим одну и ту же переменную с разными значениями: сделать(
        опознать(текст, "Rammstein"),
        вывод(текст),
        изменить(текст, "Гражданская Оборона"),
        вывод(текст)
    )
